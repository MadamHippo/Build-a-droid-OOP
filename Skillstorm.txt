Quiz 1:


Identify the correct signature of the main method by selecting the appropriate codes. (Select all that apply)
  public static void main(String[] args) 
  static public void main(String[] args) 
  
  
Explaination:

public static void main(String[] args) - This is the correct and standard signature for the main method.

static public void main(String[] args) - This is also a valid signature for the main method. The placement of the "static" and "public" keywords can be interchanged without affecting the functionality.
